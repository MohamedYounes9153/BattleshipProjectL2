Groupe : BENHARRATS Nadir, YOUNES Mohamed


Le code suivant possède 2 mode de jeu :
	- -f/cat : Lecture d'un fichier permettant de visualiser une partie dans un tableau de taille quelconque, avec navires de tailles quelconques.
		   Possibilité de se déplacer (attention aux collisions et aux mines !)
		   Partie se déroulant sur un seul tableau pour 2 joueurs.



	- -i : Partie interractive contre un bot jouant aléatoirement dans une partie suivant les règles originales de la bataille navale :
	        	* Plateau 10x10
	        	* 5 navires : un de taille 5, un de taille 4, 2 de taille 3 et 1 de taille 2
	        	* Possibilité de déplacer ses navires (attention aux collisions et aux mines !)
	        	* Le joueur et le bot possèdent chacun leur propre tableau, et le joueur possède également un tableau affichant ses tirs sur le tableau adverse (sans y afficher les navires).
